# سیستم مدیریت محتوای فانوس
> سیستم مدیریت محتوای فانوس برپایه فریم ورک لاراول می باشد.


# ویژگی ها
- برپایه فریم ورک لاراول <small>5.8</small>
- سیستم ماژولار
- مدیریت چندگانه تم ها
- سیستم هوک
- سیستم API
- پنل ادمین API محور
- پشتیبانی از فریم ورک های مطرح فرانت اند
  - Vuejs
  - Reactjs
  - Angularjs
